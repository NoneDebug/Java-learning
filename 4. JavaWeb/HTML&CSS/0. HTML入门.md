# **一** 简介

## HTML & CSS & JavaScript 的作用

- `HTML`：网页主体结构的搭建
- `CSS`：页面元素美化
- `JavaScript`：页面元素的动态处理

## 什么是 `HTML`

- `Hyper Text Markup Language` 缩写，超文本标记语言，用于搭建网页结构，在网页上展示内容


### 相关定义

- 超文本语言：`HTML`本质是文本语言，普通的文本文件只能显示字符，普通的文本文件只能显示字符，但是 `HTML` 技术则通过 `HTML` 标签语言把其他网页、视频、音频、图片各种多媒体资源引入到当前网页中，让网页有了非常丰富的呈现方式，这就是超文本。（本质还是文本）
- 标记语言：由一系列【标签】组成，没有变量、常量、流程控制、异常处理、IO等，每个标签都有它固定的页面展示效果。
	- 标签的分类
		- 双标签：标签是通过一组尖括号 + 标签名 的方式来定义
			- `<p> hello HTML.</p>`
			- 使用一个p标签定义一个段落，`<p>` 叫开始标签，`</p>` 叫结束标签，开始标签和结束标签一起构成了一个完整的标签，二者之间部分叫文本标签体，也简称为：标签体。
		- 单标签：
			- `<input type="text" name="username" />`
			- `input`是输入流标签，用于接收用户信息
		- 属性：
			- `<a href="http://www.xxx.com">show detail</a>`
			- `href` 网址是属性，`href`是属性名，网址是属性值


## 文档创建

- `HTML` 文件是浏览器负责解析和展示
- `HTML` 文件是纯文本文件，普通编辑工具都可以编辑


## 基础结构

- 根标签： `<html> <html>`
- 主体标签：`<body> </body>`
- 头标签：`<head></head>
	- 不显示在主体上，但是很重要的内容
	- 字符集 `<meta charset = "utf-8"/>` 告诉浏览器使用什么字符集对文件解码
	- `css` 引入
	- `js` 引入
	- `<title>第一个页面</title>` 标签页的名字
	- 其他功能代码
- 文档声明：
	- `<!DOCTYPE html>`
- 注释：
	- `<!--` 和 `-->` 中间

eg:

```
<!--
		1. html 文件的根标签 <html> </html> 所有的其他标签都必须在这个标签中间
		2. html 根标签下有两个一级子标签
			<head></head> 		头标签，不显示在页面主体上，但是很重要的内容
								(1) 字符集 <meta charset="utf-8"/> 告诉浏览器用什么字符集对文件解码
								(2) css引入
								(3) js引入
								(4) 其他功能代码
			<body></body>		体标签
-->

<!DOCTYPE html>
<html>

	<head>
			<title>第一个页面</title>
			<meta charset="utf-8"/> 
	</head>
	<body>
			<h1>热词里的中国活力 | 今年中央一号文件里的这些热词，你了解吗？</h1>
	</body>


</html>
```

## 专业词汇

- 标签 tag，页面的一对 `<>`
- 属性 attribute，对标签特征进行设置的一种方式，一般在开始标签中定义
	- e.g. `<input type="password"/>` 表示此类型为密码类型，是一种用户输入的数据
- 文本 text
	- 双标签中的文本
	- e.g. `<h1>热词里的中国活力 | 今年中央一号文件里的这些热词，你了解吗？</h1>` 中的一堆文字称为文本
- 元素 
	- 开始标签 + 属性 + 文本 + 结束标签 称之为一个元素



## 语法细节

- 根标签只能有一个
- 无论是双标签还是单标签都需要正常关闭
- 标签可以嵌套但不能交叉嵌套
- 注释语法 `<!-- -->`，注意不能嵌套
- 属性必须有值，值必须加引号，`H5`中属性名和值相同时可以省略属性值
- `HTML`中不严格区分字符串使用单双引号
	- 想在字符串中使用单双引号，只需要单双引号交替使用即可
- `HTML` 标签不严格区分大小写，但是不能大小写混用（首尾必须同大写或同小写）
- `HTML` 中不允许自定义签名，强行自定义则无效



# **二** 常见标签

- 标题标签：`<h1> <\h1>` h1 ~ h6

- 自然段标签：`<p> <\p>` 表示该段落开始前 和 结束后都会换行

- 换行标签：在段落后加入 `<br/>` 即可
	- `<hr/>` 加入一条横线（分割线）

- 列表标签：
	- `ol` 有序列表 `1 2 3`
	- `ul` 无序列表 `· · ·` 
	- `li` 列表项
	- 列表标签可以嵌套使用

- 超链接标签
	- 定义：点击后带有链接跳转的标签，也叫做 a 标签
	- `href` 属性用于定义连接
		格式：`<a href="https://www.atguigu.com/">尚硅谷</a>`
		`href=` 后面表示点击后跳转的地址 

其它略
