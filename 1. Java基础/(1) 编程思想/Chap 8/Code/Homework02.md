## 写出四种访问修饰符各自的访问权限

|       | 同方法 | 同包  | 子类(不同包) |  其它  |
|-------| -------|-------|-------|-------|
|public  |  ✔️  |  ✔️   |  ✔️  |   ✔️  |
|protected|  ✔️ |  ✔️  |   ✔️  |    ❌   |
|  默认  |  ✔️  |  ✔️  |   ❌    |   ❌    |
| private|  ✔️  |   ❌    |   ❌    |    ❌   |

- 注意，同包的情况也包含同包的 子类 或 其它类对使用类别的成员的使用情况
